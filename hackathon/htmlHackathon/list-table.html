HTML Lists and Tables

This page gives a brief note on HTML nested lists and tables including rowspan and colspan attributes.

HTML Nested Lists

Lists are used to display related items. A nested list means placing one list inside another list item.

Example of Nested List

  • Fruits
    • Apple
    • Banana
    • Citrus
      1. Orange
      2. Lemon
  • Vegetables
    • Leafy
      • Spinach
      • Kale
    • Root
      • Carrot
      • Beetroot

HTML Tables

Tables are used to organize data into rows and columns. rowspan merges cells vertically, and colspan merges cells horizontally.

Example of Table with Rowspan and Colspan

Student Marks
Name Marks
Math Science
Alex 85 90
Maria 88 92
John Absent

Notes

  • Use lists to group items logically. Nested lists help represent hierarchy.
  • Tables should be used only for tabular data.
  • rowspan and colspan help to merge table cells.
