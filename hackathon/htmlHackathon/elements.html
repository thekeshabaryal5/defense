HTML Elements

HTML stands for HyperText Markup Language. It is the standard language used to create and structure the content of a webpage.

  • HyperText: Ability to create links connecting web pages together.
  • Markup Language: Use "tags" to add meaning and structure to content.

First Website: https://info.cern.ch/hypertext/WWW/TheProject.html

1. The Heading (<h1> to <h6>)

Defines titles and subtitles for logical hierarchy of content.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

2. The Paragraph (<p>)

Defines blocks of text. Browsers add spacing before and after paragraphs. 

<p>This is the first paragraph.</p>
<p>This is a second, separate paragraph.</p>

3. HTML Horizontal Rule (<hr>)

Defines a thematic break, often shown as a horizontal line.

<p>This is the first topic.</p>
<hr />
<p>This is a different topic.</p>

4. The Line Break (<br>)

Breaks text onto the next line, like pressing Enter once.

<p>221B Baker Street<br>
London, England</p>

5. List Elements (<ul>, <ol>, <li>)

Used to group related items in bulleted (unordered) or numbered (ordered) lists. 

<ul>
  <li>Tea Bag</li>
  <li>Water</li>
</ul>

<ol>
  <li>Boil water</li>
  <li>Add tea bag</li>
</ol>

6. The Anchor Tag (<a>)

Creates hyperlinks. Requires href attribute.

<a href="https://www.coderarmy.in/">Visit Coder Army</a>

Target attribute:

  • _self → Default, same tab
  • _blank → Opens in new tab

7. The Image Tag (<img>)

Embeds images into a webpage. Requires src and alt attributes. 

<img src="dog.jpg" alt="This is Dog Photo" height="300px" width="300px">
